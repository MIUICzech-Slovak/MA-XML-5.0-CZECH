What are Xiaomi NFC tags?

NFC Tags are a function of NFC when the NFC-enabled mobile phone touch labels, mobile phones will be able to adjust the settings, launch applications or perform specific actions according to label instructions, such as opening and closing WiFi, enter silent mode, dial a specific phone.

You can set application with NFC tag the function to change at any time after touching the label instructions executed.
